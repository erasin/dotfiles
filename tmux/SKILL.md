# tmux 使用技巧

## 概述

tmux 是一个终端复用器，允许在单个终端中运行多个会话、窗口和窗格。本文档介绍如何高效使用 tmux。

## 核心概念

```
session（会话）
├── window（窗口）
│   ├── pane（窗格） 1
│   ├── pane（窗格） 2
│   └── pane（窗格） 3
└── window（窗口）
```

| 概念 | 说明 |
| ---- | ---- |
| session | 一个独立的工作环境，包含多个 window |
| window | 相当于一个终端标签页 |
| pane | window 内的分割区域 |

## 基础操作

### 会话管理

```bash
# 创建新会话
tmux new -s <session-name>

# 列出所有会话
tmux ls

# 附加到会话
tmux attach -t <session-name>

# 分离当前会话（退出 tmux 但保持运行）
# 在 tmux 内按 <prefix> d

# 关闭会话
tmux kill-session -t <session-name>
```

### 窗口操作

| 按键 | 说明 |
| ---- | ---- |
| `c` | 创建新窗口 |
| `,` | 重命名当前窗口 |
| `&` | 关闭当前窗口 |
| `0-9` | 切换到指定编号窗口 |
| `w` | 显示窗口列表 |
| `Tab` | 切换到上一个窗口 |

### 窗格操作

| 按键 | 说明 |
| ---- | ---- |
| `-` | 水平分割窗格 |
| `_` | 垂直分割窗格 |
| `x` | 关闭当前窗格 |
| `o` | 在窗格间切换 |
| `;` | 切换到上一个活跃窗格 |
| `q` | 显示窗格编号 |

## 进阶技巧

### 复制模式（vim 风格）

复制模式用于在 tmux 中选择和复制文本。

```bash
# 进入复制模式
[

# 导航（vim 风格）
h/j/k/l    左/下/上/右移动
w/b        按单词前进/后退
0/$        行首/行尾
gg/G       文件开头/结尾

# 选择
v          开始选择
V          按行选择
C-v        矩形选择

# 复制
y          复制选中内容
Enter      复制并退出

# 粘贴
]          粘贴
```

### 同步窗格

在多个窗格同时执行相同命令：

```bash
# 在 tmux 命令模式（<prefix> :）输入
setw synchronize-panes on

# 关闭同步
setw synchronize-panes off
```

### 自定义布局

| 按键 | 说明 |
| ---- | ---- |
| `M-1` | 水平均匀分布 |
| `M-2` | 垂直均匀分布 |
| `M-3` | 主窗格占 2/3，其他均分 |
| `M-5` | 主窗格占一半，其他均分 |
| `Space` | 循环切换预设布局 |

### 窗格大小调整

```bash
# 快速调整（按住 <prefix>）
<C-h/j/k/l>  调整大小 2 格

# 精确调整（在命令模式）
resize-pane -L 10    # 向左扩展 10 格
resize-pane -R 10    # 向右扩展
resize-pane -U 5     # 向上
resize-pane -D 5     # 向下

# 最大化/还原窗格
f
```

## 高效工作流

### 场景一：远程开发

```bash
# 1. 创建开发会话
tmux new -s dev

# 2. 分割窗格布局
#    +------------------+------------------+
#    |     editor       |      shell       |
#    |                  |                  |
#    +------------------+------------------+
#    |        output                    |
#    +----------------------------------+

# 3. 在编辑器窗口运行 vim/neovim
# 4. 在 shell 窗口运行测试/构建
# 5. 分离会话，关闭终端
# 6. 稍后重新附加继续工作
```

### 场景二：多服务器管理

```bash
# 1. 创建主会话
tmux new -s servers

# 2. 创建多个窗口连接不同服务器
# window 1: 生产环境
# window 2: 测试环境
# window 3: 开发环境

# 3. 使用 synchronize-panes 同时查看多台服务器状态
```

### 场景三：临时任务

```bash
# 使用 popup scratch 终端
# 按 v 打开临时终端，关闭即消失
# 适合临时运行命令、查看日志
```

### 场景四：会话持久化

配合 `tmux-resurrect` 和 `tmux-continuum` 插件：

```bash
# 保存会话快照
<prefix Ctrl-s>

# 恢复会话
<prefix Ctrl-r>
```

## 故障排除

### tmux 无响应

```bash
# 强制重载配置
<prefix> : source-file ~/.tmux.conf

# 或在 shell 中
tmux source-file ~/.tmux.conf
```

### 复制模式无法选择文本

确保进入了复制模式（按 `[`），然后再使用 vim 风格导航。

### 终端显示异常

```bash
# 检查 256 色支持
tmux set -g default-terminal "screen-256color"

# 检查更新
tmux display-message
```

### 插件不工作

```bash
# 重新安装插件
<prefix> I  (大写 I)
```
