# tmux

prefix <C-a>, 默认 <C-b>

使用方式 <C-a> 后键入下方的key

# session

<C-c> | 新建 session
<C-f> | 查询 session
d | quit

# window

| key | -r    | cmd                   |
| --: | ----- | --------------------- |
| `c` |       | 新建窗口              |
| `w` |       | 窗口列表              |
| `e` |       | 切换显示 tab          |
| `n` | <C-h> | 下一个窗口            |
| `p` | <C-l> | 上一个窗口            |
| `.` |       | 交换窗口序号          |
| ',' |       | 重命名窗口标题        |
| `a` | <Tab> | 最后的窗口            |
| 0-9 |       | select-window -t :0-9 |

# pane

| key | -r  | cmd            |
| --: | --- | -------------- |
| `-` |     | 列窗格 h pane  |
| `_` |     | 行窗格 v pane  |
| `h` |     | 切换窗格       |
| `j` |     | 切换窗格       |
| `k` |     | 切换窗格       |
| `l` |     | 切换窗格       |
| `H` |     | 扩充窗格尺寸   |
| `J` |     | 扩充窗格尺寸   |
| `K` |     | 扩充窗格尺寸   |
| `L` |     | 扩充窗格尺寸   |
| `x` |     | 关闭           |
| `\`` |     | 切换窗格最大化 |
